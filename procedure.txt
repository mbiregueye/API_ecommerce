-> npm init -y

2- installer le depedances

-> yarn add express mongoose dotenv nodemon  dans ce cas il va falloir installer yarn au paravant avec -> npm install --global yarn

on peux aussi utiliser npm i -s express mongoose dotenv nodemaon

3- Dans le fichier package.json changer la ligne  "test": "echo \"Error: no test specified\" && exit 1"    en   "start": "nodemon index.js" 
4- Utliser la commande
-> yarn start pour demarer maintenat le projet on peux aussi utiliser -> npm start

6- aller dans https://cloud.mongodb.com/ pour creer un compte gratuit car nous utliserons la base de donnes NoSQL MongoDB cloud

9- ceer le dossier model et les differents model Userjs, Order.js, Cart.js et configurer es differents propriets de chaque Schema ou table ou document

10- dans router on creer un auth.js pour la l'authentification et la creation de compte
